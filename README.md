# Alethia Live

Phase 0 starter scaffold for a live healthcare navigation and health literacy agent.

## What this starter includes
- single Next.js app structure
- placeholder API routes
- starter UI components
- safety prompt files
- sample document images
- docs folder for spec, architecture, and demo assets

## Quick start
```bash
cp .env.local.example .env.local
npm install
npm run dev
```

## Notes
- `src/app/api/live-token/route.ts` is a placeholder.
- `src/app/api/summary/route.ts` currently returns mock structured data.
- This is the scaffold, not the finished Gemini Live integration.
