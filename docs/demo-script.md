# Alethia Live — Demo Script

## Demo Goal
Show a safe, real-time, multimodal healthcare navigation agent that helps users understand next steps without diagnosing or prescribing.

## Demo Structure
The demo will show two short scenarios.

---

## Scenario 1 — Care Navigation

### User Input
“I’ve had a fever and sore throat for two days. I’m not sure if I should wait, book an appointment, or go somewhere urgent.”

### What the Agent Should Do
- ask 2 to 3 clarifying questions
- speak naturally in response
- avoid diagnosis
- summarize the concern
- suggest a high-level care-path category
- list questions to ask a clinician
- mention emergency red flags
- keep the disclaimer visible

### What to Highlight in the Demo
- live voice interaction
- cautious safety framing
- real-time transcript
- useful structured summary cards

---

## Scenario 2 — Document Explanation

### User Action
Upload a screenshot/photo of an after-visit summary or appointment instruction page.

### What the Agent Should Do
- explain the document in plain English
- highlight key points
- identify next-step questions
- avoid medical overreach
- maintain the visible disclaimer

### What to Highlight in the Demo
- multimodal input
- health literacy support
- practical usefulness
- strong scope control

## Key Talking Points
- This is not a diagnostic system.
- This is a navigation and health literacy companion.
- It helps users prepare for clinician conversations.
- It uses Gemini Live for real-time multimodal interaction.
- It is deployed on Google Cloud.
